Elaborar Banco de Dados

Análise

Finalidade

Construir um banco de dados através da documentação da arquitetura do software contendo diagramas UML (casos de uso, classe, interação, implantação, componentes e comportamento) entidade-relacionamento e relacional. Para tanto, foi preciso usar os artefatos gerados na etapa anterior, principalmente, os requisitos. Neste sentido, também elaborada uma documentação dos requisitos atualizados, pois, com a discussão do projeto e novas reuniões com o cliente, alguns requisitos, em geral, foram alterados.

Observações

Identificar as reais necessidades do seu projeto e buscar as ferramentas que melhor supram essas necessidades.
